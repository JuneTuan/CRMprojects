## 1. 系统概述

### 1.1 系统简介

JunLite CRM V6.0 是一个功能完善的客户关系管理系统，提供Web管理端和H5移动端双端支持，帮助企业高效管理客户、产品、订单、营销活动、销售线索等业务。

### 1.2 核心功能

#### 管理端功能
- 📊 **仪表板**：实时数据概览、统计分析
- 👥 **客户管理**：客户信息管理、积分管理、排序分页
- 📦 **产品管理**：产品信息、库存管理
- 📝 **订单管理**：订单处理、支付管理、订单项管理
- 🎫 **优惠券管理**：优惠券发放、核销管理
- 🎯 **活动管理**：营销活动、抽奖游戏
- 🎁 **抽奖管理**：奖品配置、抽奖记录
- 👨‍💼 **员工管理**：员工信息、角色权限
- 📈 **统计分析**：销售统计、客户分析
- 📋 **审计日志**：登录记录、操作记录、安全审计
- 🏆 **会员等级**：等级配置、自动升级、等级变更历史
- 🔄 **线索管理**：线索录入、分配、跟进、转化 (V6.0新增)
- 📖 **字典管理**：枚举值配置、客户来源管理 (V6.0新增)

#### H5前端功能
- 🔐 **登录注册**：用户认证、自动标注来源
- 🏠 **首页**：活动展示、快捷入口
- 🎮 **抽奖游戏**：参与抽奖活动、赢取奖品
- 💳 **我的优惠券**：优惠券管理、使用记录
- ⭐ **积分记录**：积分查询、积分历史
- 🛒 **购买记录**：订单查询、订单详情
- 🏆 **抽奖记录**：抽奖历史、奖品领取
- 👤 **个人中心**：个人信息、设置

### 1.3 V6.0 新增功能

#### 线索管理模块
- 线索录入与管理
- 线索分配给销售人员
- 线索跟进记录管理
- 线索评分系统
- 线索转化为客户和订单

#### 字典管理模块
- 可配置的枚举值管理
- 客户来源动态配置
- 支持多类型字典

#### 客户管理增强
- 支持表格排序
- 支持分页浏览
- 来源字段使用字典值

#### H5注册优化
- 自动标注注册来源
- 支持字典来源配置

## 2. 用户角色与权限

### 2.1 角色定义

系统支持以下角色：

| 角色 | 描述 | 权限级别 |
|------|------|----------|
| **超级管理员** | 系统最高权限 | 全部权限 |
| **管理员** | 业务管理权限 | 大部分权限 |
| **员工** | 基础操作权限 | 有限权限 |

### 2.2 权限说明

#### 超级管理员权限
- ✅ 所有模块的完整访问权限
- ✅ 用户管理（创建、编辑、删除）
- ✅ 角色权限配置
- ✅ 系统设置
- ✅ 数据导出
- ✅ 字典管理

#### 管理员权限
- ✅ 客户管理（查看、编辑）
- ✅ 产品管理（完整权限）
- ✅ 订单管理（完整权限）
- ✅ 优惠券管理（完整权限）
- ✅ 活动管理（完整权限）
- ✅ 抽奖管理（完整权限）
- ✅ 线索管理（完整权限）
- ✅ 统计分析（查看权限）
- ❌ 用户管理
- ❌ 角色权限配置

#### 员工权限
- ✅ 客户管理（查看权限）
- ✅ 产品管理（查看权限）

- ✅ 优惠券核销
- ✅ 抽奖记录查看
- ✅ 线索管理（查看、编辑）

## 3. 管理端使用指南

### 3.1 登录系统

#### 步骤说明

1. 打开浏览器，访问管理端地址：`http://localhost:8080`
2. 输入用户名和密码
3. 点击"登录"按钮
4. 首次登录建议修改密码

### 3.2 仪表板

#### 功能说明

仪表板提供系统核心数据的实时概览：

- **客户总数**：系统中的客户数量
- **订单总数**：所有订单的数量
- **销售总额**：累计销售金额
- **活动数量**：进行中的活动数量
- **图表展示**：销售趋势、客户增长、等级分布

### 3.3 客户管理

#### 3.3.1 客户列表

**功能说明**
- 查看所有客户信息
- 搜索和筛选客户
- 支持排序（按姓名、注册时间、消费金额等）
- 支持分页浏览
- 查看客户详情
- 编辑客户信息
- 删除客户

**操作步骤**

1. 点击左侧菜单"客户管理"
2. 查看客户列表
3. 点击表头可进行排序
4. 使用分页器浏览更多数据
5. 使用搜索框输入客户姓名、手机号等关键词
6. 点击"查看"按钮查看客户详情
7. 点击"编辑"按钮修改客户信息

**客户列表字段**

| 字段 | 说明 |
|------|------|
| 客户ID | 系统自动生成的唯一标识 |
| 客户姓名 | 客户的真实姓名 |
| 手机号 | 客户的联系电话 |
| 邮箱 | 客户的电子邮箱 |
| 来源 | 客户来源（字典配置） |
| 会员等级 | 当前会员等级 |
| 积分余额 | 当前积分数量 |
| 累计消费 | 总消费金额 |
| 注册时间 | 账号注册时间 |

#### 3.3.2 添加客户

**操作步骤**

1. 点击"添加客户"按钮
2. 填写客户信息：
   - 客户姓名（必填）
   - 手机号（必填）
   - 邮箱（选填）
   - 来源（从字典选择）
   - 地址（选填）
   - 备注（选填）
3. 点击"保存"按钮

### 3.4 线索管理 (V6.0新增)

#### 3.4.1 线索列表

**功能说明**
- 查看所有销售线索
- 搜索和筛选线索
- 查看线索详情
- 编辑线索信息
- 删除线索
- 分配线索给销售人员

**操作步骤**

1. 点击左侧菜单"线索管理"
2. 查看线索列表
3. 使用搜索框输入线索姓名、手机号等关键词
4. 使用筛选器按状态、来源等条件筛选
5. 点击"查看"按钮查看线索详情
6. 点击"编辑"按钮修改线索信息
7. 点击"分配"按钮将线索分配给销售人员

**线索列表字段**

| 字段 | 说明 |
|------|------|
| 线索ID | 系统自动生成的唯一标识 |
| 线索姓名 | 线索的姓名 |
| 手机号 | 线索的联系电话 |
| 邮箱 | 线索的电子邮箱 |
| 公司 | 线索所属公司 |
| 来源 | 线索来源渠道 |
| 状态 | 新线索/跟进中/已转化/已关闭 |
| 优先级 | 高/中/低 |
| 评分 | 线索评分（0-100） |
| 分配给 | 负责的销售人员 |
| 创建时间 | 线索创建时间 |

#### 3.4.2 添加线索

**操作步骤**

1. 点击"添加线索"按钮
2. 填写线索信息：
   - 线索姓名（必填）
   - 手机号（必填）
   - 邮箱（选填）
   - 公司（选填）
   - 职位（选填）
   - 来源（从字典选择）
   - 优先级（高/中/低）
   - 线索描述（选填）
3. 点击"保存"按钮

#### 3.4.3 线索分配

**操作步骤**

1. 在线索列表中，点击"分配"按钮
2. 选择要分配的销售人员
3. 点击"确认"按钮
4. 系统自动通知销售人员

#### 3.4.4 线索跟进

**功能说明**
- 添加跟进记录
- 查看跟进历史
- 更新线索状态

**操作步骤**

1. 在线索详情页面，点击"添加跟进"
2. 填写跟进信息：
   - 跟进方式（电话/邮件/面谈/其他）
   - 跟进内容（必填）
   - 下次跟进时间（选填）
3. 点击"保存"按钮

#### 3.4.5 线索转化

**功能说明**
- 将线索转化为客户
- 同时创建订单
- 自动更新线索状态

**操作步骤**

1. 在线索详情页面，点击"转化为订单"按钮
2. 系统自动跳转到订单管理页面
3. 线索客户信息自动填充
4. 添加订单产品
5. 点击"保存"按钮创建订单
6. 系统自动：
   - 创建客户账号
   - 更新线索状态为"已转化"
   - 记录转化金额

### 3.5 字典管理 (V6.0新增)

#### 3.5.1 字典列表

**功能说明**
- 管理系统枚举值
- 配置客户来源选项
- 支持多类型字典

**操作步骤**

1. 点击左侧菜单"字典管理"
2. 查看字典列表
3. 使用筛选器按字典类型筛选
4. 点击"添加"按钮新增字典项
5. 点击"编辑"按钮修改字典项
6. 点击"删除"按钮删除字典项

**字典字段**

| 字段 | 说明 |
|------|------|
| 字典ID | 系统自动生成的唯一标识 |
| 字典类型 | 字典分类（如：customer_source） |
| 字典标签 | 显示名称 |
| 字典值 | 存储值 |
| 排序 | 显示顺序 |
| 状态 | 启用/禁用 |

#### 3.5.2 添加字典项

**操作步骤**

1. 点击"添加字典"按钮
2. 填写字典信息：
   - 字典类型（必填）：如 customer_source
   - 字典标签（必填）：如 "官网注册"
   - 字典值（必填）：如 "website_register"
   - 排序（选填）：数字越小越靠前
   - 状态：启用/禁用
3. 点击"保存"按钮

**预设字典类型**

| 字典类型 | 说明 | 预设值 |
|----------|------|--------|
| customer_source | 客户来源 | 官网注册、H5注册、电话咨询、线下门店、转介绍、其他 |
| lead_source | 线索来源 | 官网表单、电话咨询、展会、广告投放、转介绍、其他 |
| lead_priority | 线索优先级 | 高、中、低 |

## 4. H5前端使用指南

### 4.1 用户注册

#### 步骤说明

1. 打开H5前端：`http://localhost:8081`
2. 点击"注册"按钮
3. 填写注册信息：
   - 用户名（必填）
   - 密码（必填）
   - 确认密码（必填）
   - 姓名（必填）
   - 手机号（必填）
   - 邮箱（选填）
4. 点击"注册"按钮
5. 注册成功后自动登录

**注意事项**
- 用户名必须唯一
- 密码长度至少6位
- 手机号格式：11位数字
- 注册来源自动标注为"H5注册"

### 4.2 用户登录

#### 步骤说明

1. 打开H5前端：`http://localhost:8081`
2. 输入用户名和密码
3. 点击"登录"按钮
4. 登录成功后进入首页

### 4.3 首页

#### 功能说明

首页展示当前进行中的活动和快捷入口：

- **活动列表**：显示所有进行中的活动
- **快捷入口**：快速访问常用功能
- **个人信息**：显示当前用户的基本信息

### 4.4 抽奖游戏

#### 参与活动

1. 在首页选择一个活动
2. 点击"参与"按钮
3. 查看活动详情
4. 点击"开始游戏"按钮

#### 抽奖规则

- 每个活动有固定的免费抽奖次数
- 超过免费次数需要消耗积分
- 积分消耗根据活动配置而定
- 每个用户有最大抽奖次数限制

### 4.5 我的优惠券

#### 功能说明

- 查看所有优惠券
- 查看优惠券详情

## 6. 常见问题

### Q1: 忘记密码怎么办？

A: 请联系管理员重置密码。

### Q2: 线索转化后可以撤销吗？

A: 线索转化后不可撤销，但可以删除创建的订单和客户。

### Q3: 如何添加新的客户来源选项？

A: 在字典管理中添加 customer_source 类型的字典项。

### Q4: H5注册的客户来源是什么？

A: H5注册的客户来源自动标注为"h5_register"。

### Q5: 线索评分如何计算？

A: 线索评分由销售人员手动设置，范围0-100分。
