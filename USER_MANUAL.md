## 1. 系统概述

### 1.1 系统简介

JunLite CRM V5.2R1 是一个功能完善的客户关系管理系统，提供Web管理端和H5移动端双端支持，帮助企业高效管理客户、产品、订单、营销活动等业务。

### 1.2 核心功能

#### 管理端功能
- 📊 **仪表板**：实时数据概览、统计分析
- 👥 **客户管理**：客户信息管理、积分管理
- 📦 **产品管理**：产品信息、库存管理
- 📝 **订单管理**：订单处理、支付管理
- 🎫 **优惠券管理**：优惠券发放、核销管理
- 🎯 **活动管理**：营销活动、抽奖游戏
- 🎁 **抽奖管理**：奖品配置、抽奖记录
- 👨‍💼 **员工管理**：员工信息、角色权限
- 📈 **统计分析**：销售统计、客户分析
- 📋 **审计日志**：登录记录、操作记录、安全审计
- 🏆 **会员等级**：等级配置、自动升级、等级变更历史

#### H5前端功能
- 🔐 **登录注册**：用户认证、账号管理
- 🏠 **首页**：活动展示、快捷入口
- 🎮 **抽奖游戏**：参与抽奖活动、赢取奖品
- 💳 **我的优惠券**：优惠券管理、使用记录
- ⭐ **积分记录**：积分查询、积分历史
- 🛒 **购买记录**：订单查询、订单详情
- 🏆 **抽奖记录**：抽奖历史、奖品领取
- 👤 **个人中心**：个人信息、设置管理

### 1.3 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                    JunLite CRM V5.2R1                    │
├─────────────────────────────────────────────────────────────┤
│  管理端 (Web)          后端服务 (NestJS)    H5前端 (UniApp)│
│  - Vue 3              - RESTful API      - Vue 3          │
│  - Element Plus        - MySQL 8.0        - 跨平台支持     │

└─────────────────────────────────────────────────────────────┘
```

## 2. 用户角色与权限

### 2.1 角色定义

系统支持以下角色：

| 角色 | 描述 | 权限级别 |
|------|------|----------|
| **超级管理员** | 系统最高权限 | 全部权限 |
| **管理员** | 业务管理权限 | 大部分权限 |
| **员工** | 基础操作权限 | 有限权限 |

### 2.2 权限说明

#### 超级管理员权限
- ✅ 所有模块的完整访问权限
- ✅ 用户管理（创建、编辑、删除）
- ✅ 角色权限配置
- ✅ 系统设置
- ✅ 数据导出

#### 管理员权限
- ✅ 客户管理（查看、编辑）
- ✅ 产品管理（完整权限）
- ✅ 订单管理（完整权限）
- ✅ 优惠券管理（完整权限）
- ✅ 活动管理（完整权限）
- ✅ 抽奖管理（完整权限）
- ✅ 统计分析（查看权限）
- ❌ 用户管理
- ❌ 角色权限配置

#### 员工权限
- ✅ 客户管理（查看权限）
- ✅ 产品管理（查看权限）
- ✅ 订单管理（查看、编辑）
- ✅ 优惠券核销
- ✅ 抽奖记录查看
- ❌ 系统配置

## 3. 管理端使用指南

### 3.1 登录系统

#### 步骤说明

1. 打开浏览器，访问管理端地址：`http://localhost:8080`
2. 输入用户名和密码
3. 点击"登录"按钮
4. 首次登录建议修改密码

#### 登录界面

```
┌─────────────────────────────────┐
│                               │
│        JunLite CRM             │
│                               │
│   用户名: [______________]     │
│   密码:   [______________]     │
│                               │
│      [ 登录 ]  [ 重置 ]       │
│                               │
└─────────────────────────────────┘
```

### 3.2 仪表板

#### 功能说明

仪表板提供系统核心数据的实时概览：

- **客户总数**：系统中的客户数量
- **订单总数**：所有订单的数量
- **销售总额**：累计销售金额
- **活动数量**：进行中的活动数量
- **图表展示**：销售趋势、客户增长、等级分布

#### 使用方法

1. 登录后自动进入仪表板
2. 查看各项数据指标
3. 点击图表可查看详细数据
4. 使用时间筛选器查看不同时间段的数据

### 3.3 客户管理

#### 3.3.1 客户列表

**功能说明**
- 查看所有客户信息
- 搜索和筛选客户
- 查看客户详情
- 编辑客户信息
- 删除客户

**操作步骤**

1. 点击左侧菜单"客户管理"
2. 查看客户列表
3. 使用搜索框输入客户姓名、手机号等关键词
4. 使用筛选器按等级、注册时间等条件筛选
5. 点击"查看"按钮查看客户详情
6. 点击"编辑"按钮修改客户信息
7. 点击"删除"按钮删除客户

**客户列表字段**

| 字段 | 说明 |
|------|------|
| 客户ID | 系统自动生成的唯一标识 |
| 客户姓名 | 客户的真实姓名 |
| 手机号 | 客户的联系电话 |
| 邮箱 | 客户的电子邮箱 |
| 会员等级 | 当前会员等级 |
| 积分余额 | 当前积分数量 |
| 累计消费 | 总消费金额 |
| 注册时间 | 账号注册时间 |

#### 3.3.2 添加客户

**操作步骤**

1. 点击"添加客户"按钮
2. 填写客户信息：
   - 客户姓名（必填）
   - 手机号（必填）
   - 邮箱（选填）
   - 地址（选填）
   - 备注（选填）
3. 点击"保存"按钮

**注意事项**
- 手机号必须唯一
- 手机号格式：11位数字
- 邮箱格式必须正确

#### 3.3.3 积分管理

**功能说明**
- 为客户增加积分
- 扣除客户积分
- 查看积分历史记录

**操作步骤**

1. 在客户详情页面，点击"积分管理"
2. 选择操作类型：
   - 增加积分：输入增加的积分数和原因
   - 扣除积分：输入扣除的积分数和原因
3. 点击"确认"按钮
4. 查看积分历史记录

**积分规则**
- 1元消费 = 1积分
- 积分可用于兑换优惠券
- 积分可用于参与抽奖活动

#### 3.4.1 产品列表

5. 点击"查看"按钮查看产品详情

**产品列表字段**

| 字段 | 说明 |
|------|------|

#### 3.4.2 添加产品

**操作步骤**

1. 点击"添加产品"按钮
2. 填写产品信息：
   - 产品名称（必填）
   - 产品编号（必填，唯一）
   - 分类（必填）
   - 价格（必填）
   - 库存（必填）
   - 产品描述（选填）

**注意事项**
- 产品编号必须唯一
- 价格必须大于0
- 库存必须大于等于0

#### 3.4.3 库存管理

**操作步骤**

1. 在产品详情页面，点击"库存管理"
2. 输入调整数量：
   - 正数表示增加库存
   - 负数表示减少库存
3. 输入调整原因
4. 点击"确认"按钮

2. 查看订单列表
3. 使用搜索框输入订单号、客户姓名等关键词
4. 使用筛选器按状态、时间等条件筛选
5. 点击"查看"按钮查看订单详情
6. 点击"支付"按钮处理订单支付
7. 点击"取消"按钮取消订单

**订单列表字段**

| 字段 | 说明 |
|------|------|
| 订单ID | 系统自动生成的唯一标识 |

#### 3.5.2 订单详情

**查看订单详情**

1. 在订单列表中，点击"查看"按钮
2. 查看订单基本信息：
   - 订单号
   - 客户信息
   - 订单金额
   - 订单状态
   - 下单时间
3. 查看订单明细：
   - 产品名称
   - 产品数量
   - 单价
   - 小计

**处理订单支付**

1. 在订单详情页面，点击"支付"按钮
2. 确认支付信息
3. 点击"确认支付"按钮
4. 系统自动更新订单状态为"已支付"
5. 系统自动增加客户积分

**取消订单**

1. 在订单详情页面，点击"取消"按钮
2. 输入取消原因
3. 点击"确认取消"按钮
4. 系统自动更新订单状态为"已取消"

## 5. 会员等级系统

### 5.1 等级说明

系统支持以下会员等级：

| 等级 | 代码 | 最低消费 | 权益说明 |
|------|------|----------|----------|
| 普通会员 | normal | 0元 | 新用户专享 |
| 白银会员 | silver | 1000元 | 专属客服 \| 生日礼包 |
| 黄金会员 | gold | 5000元 | 专属客服 \| 生日礼包 \| 优先发货 |
| 钻石会员 | diamond | 10000元 | 专属客服 \| 生日礼包 \| 优先发货 \| 专属折扣 |

### 5.2 等级升级

#### 自动升级

- 系统根据累计消费自动升级
- 订单支付成功后自动检查等级
- 满足升级条件立即升级
- 升级后发送通知

#### 升级规则

- 累计消费 = 订单总金额
- 积分按100:1换算为消费金额
- 等级只升不降（MVP版本）
- 退款不影响等级

#### 查看升级进度

1. 进入个人中心
2. 查看当前等级
3. 查看升级进度条
4. 查看距离下一等级还需消费金额

### 5.3 等级权益

#### 普通会员权益
- 基础购物功能
- 积分累计
- 参与活动

#### 白银会员权益
- 专属客服
- 生日礼包
- 积分倍数：1.2倍
- 额外抽奖次数：1次

#### 黄金会员权益
- 专属客服
- 生日礼包
- 优先发货
- 积分倍数：1.5倍
- 额外抽奖次数：3次
- 折扣率：95折

#### 钻石会员权益
- 专属客服
- 生日礼包
- 优先发货
- 专属折扣
- 积分倍数：2.0倍
- 额外抽奖次数：5次
- 折扣率：90折

### 5.4 等级管理（管理端）

#### 查看客户等级

1. 进入客户管理
2. 查看客户列表
3. 查看客户的会员等级
4. 点击"等级详情"查看等级信息

#### 手动调整等级

1. 进入客户详情
2. 点击"调整等级"按钮
3. 选择新的等级
4. 输入调整原因
5. 点击"确认"按钮

**注意事项**
- 手动调整需要填写原因
- 手动调整会记录操作日志
- 手动调整不改变消费金额

#### 查看等级变更日志

1. 进入客户详情
2. 点击"等级日志"按钮
3. 查看所有等级变更记录
4. 查看变更类型、时间、原因
